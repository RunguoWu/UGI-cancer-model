This folder contains code files to clean the CPRD and linkage data and create the study population. 

The master file "Master_create_study_population.R" is the mail code file to create the study population.

The function file "fn_clean.R" contains all functions used in the process.

The "Multiple_imputation.R" file is for imputing missing stage values after creating the study population.

Other code files are for cleaning data for different purposes, and their output files are loaded in the master file.
